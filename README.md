# Customer Sales Analytics Dashboard

## Overview
Analysed 1,642 customer transactions 
using MySQL and Excel to identify 
revenue trends and customer segments.

## Tools Used
- MySQL (Data Analysis)
- Microsoft Excel (Dashboard)

## Key Findings
- Tablet: Top revenue product (16.3%)
- Total Revenue: ₹32.6 Lakhs
- 94% customers are Low Value
- Peak sales: November 2023

- SQL_Queries.sql — All SQL queries
